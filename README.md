# 🔥 IronPulse - AI-Powered Gym Management App

## 🚀 Modern Mobile-First Gym Experience

A premium, futuristic gym management web app built with React.js, featuring gamification, AI coaching, and social features. Designed to feel like a real mobile app with smooth animations and addictive interactions.

## ✨ Features

### 🎮 Gamification System
- **XP & Levels**: Earn XP for every workout, level up, unlock achievements
- **Streak System**: Animated flame streak tracker with daily motivation
- **Badges & Achievements**: Collect badges like "Beast Mode", "Cardio Demon", "Protein Predator"
- **Battle System**: Challenge friends in push-up battles, calorie burns, and more
- **Aura System**: Daily AI-generated status like "Main Character Energy", "Villain Arc Activated"

### 🤖 AI Features
- **Funny AI Coach**: Motivational (and sarcastic) messages based on your activity
- **Mood-Based Workouts**: Select your mood, get personalized workout recommendations
- **Smart Recommendations**: AI analyzes your patterns and suggests optimal workouts

### 💪 Workout Features
- **Swipe Interface**: Tinder-style workout discovery (swipe right to save, left to skip, up to start)
- **Video Demonstrations**: YouTube embedded exercise tutorials
- **Personal Records**: Track your PRs for bench press, deadlift, squat, running
- **Workout History**: Complete attendance tracking with heatmap calendar

### 📅 Class Management
- **Live Schedule**: Browse and book classes with real-time capacity tracking
- **Trainer Profiles**: View trainer specialties, ratings, and certifications
- **Class Details**: Full information including equipment needed, difficulty, calories

### 👥 Social Features
- **Social Feed**: Share PRs, workout selfies, and achievements
- **Likes & Comments**: Engage with the gym community
- **Leaderboards**: Compete with other members

### 🎨 Design Features
- **Dark Theme**: Black background with orange gradient accents
- **Glassmorphism**: Modern glass-effect cards
- **Neon Glow Effects**: Premium glowing animations
- **Smooth Animations**: Framer Motion powered transitions
- **Mobile-First**: Optimized for mobile with bottom navigation
- **Skeleton Loading**: Professional loading states
- **Confetti Effects**: Celebration animations for achievements

## 🛠️ Tech Stack

- **React.js 19** - Latest React with hooks
- **Vite** - Lightning fast build tool
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Smooth animations
- **React Router** - Client-side routing
- **Lucide React** - Beautiful icons
- **Context API** - State management
- **LocalStorage** - Data persistence

## 📦 Installation

```bash
# Navigate to project directory
cd gym-app

# Install dependencies (already done)
npm install

# Start development server
npm run dev
```

### Key Interactions

1. **Start Workout**: Home → "Start Workout" → Select Mood → Choose Workout
2. **Browse Classes**: Schedule → Select Class → View Details → Book
3. **Swipe Workouts**: Workouts → Swipe cards (left/right/up)
4. **View Progress**: Profile → Personal Records / Attendance
5. **Challenge Friends**: Profile → Battles → View Active Battles
6. **Social Feed**: Profile → Social Feed → Like & Comment
